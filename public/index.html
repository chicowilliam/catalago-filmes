Sobre o projeto

Catálogo full stack inspirado em streaming, com foco em UX, integração com API e arquitetura limpa.
Portfolio case

Interface com linguagem de streaming, arquitetura leve e foco em sensação de produto final.

Este projeto combina front-end em JavaScript vanilla com back-end em Node.js + Express, autenticação por sessão, integração com TMDB e uma experiência visual construída para transmitir clareza, ritmo e acabamento premium.

UX focada em catálogo TMDB via backend Sessões e favoritos
Front-end

HTML, CSS e JavaScript com componentes visuais customizados, navegação por filtros, modal e microinterações.

Back-end

Express com sessões, validação, erros padronizados, integração com API externa e fallback local.

Experiência

Busca, trailers, favoritos, loading states, lazy loading e responsividade pensados como vitrine de portfólio.
